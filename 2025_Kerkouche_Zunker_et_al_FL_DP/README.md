# Federated Learning with Differential Privacy for Infectious Disease Dynamics

This repository contains code and instructions to reproduce the results of the paper:
"Federated Learning with Differential Privacy for the privacy-preserving prediction of infectious disease dynamics".

## Repository Structure

Top-level elements (selected):
- `/plots` – plotting scripts used to generate figures.
-  `/data_generation` – scripts for preparing, extending, and smoothing COVID‑19 case data at county and community level.
- `county_dp.py` –  script for running the federated learning with differential privacy on county-level data.
- `community_dp.py` - script for running the federated learning with differential privacy on community-level data.
